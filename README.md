# 📦 Sistema de Almoxarifado

Sistema de gerenciamento de almoxarifado desenvolvido em **Python** utilizando o framework **Django**, como projeto acadêmico.

## 📋 Funcionalidades

- Login de usuários
- Controle de perfis (Administrador e Funcionário)
- Cadastro de produtos
- Cadastro de categorias
- Cadastro de depósitos
- Entrada de produtos no estoque
- Saída de produtos do estoque
- Relatório por produto
- Relatório com movimentações agrupadas por depósito
- Dashboard com informações gerais

## 🛠️ Tecnologias utilizadas

- Python
- Django
- HTML5
- CSS3
- Bootstrap 5
- SQLite3
